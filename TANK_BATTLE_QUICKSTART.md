# 🎮 坦克大战 - 快速启动指南

## 🚀 快速开始

### 方法1: 直接在浏览器中打开

1. 在项目根目录找到 `tank-battle.html` 文件
2. 双击文件或右键选择"打开方式" → 选择浏览器
3. 点击"开始游戏"按钮即可开始

### 方法2: 使用本地服务器（推荐）

由于使用了ES6模块，推荐使用本地服务器运行：

```bash
# 使用Python 3
cd /Users/bingbing/demoCodex
python3 -m http.server 8000

# 或使用Node.js的http-server
npx http-server -p 8000

# 或使用PHP
php -S localhost:8000
```

然后在浏览器中访问：
- 游戏主页：`http://localhost:8000/tank-battle.html`
- 成就页面：`http://localhost:8000/tank-battle-achievements.html`

## 🎯 游戏操作

### 基本控制
- **移动**：`W` `A` `S` `D` 或 方向键
- **射击**：`空格键`
- **暂停**：`P` 或 `ESC`
- **静音**：点击右上角的音效按钮

### 游戏目标
1. 摧毁所有敌方坦克（每关20个）
2. 保护底部中央的基地（🦅）不被摧毁
3. 尽可能获得高分
4. 收集道具增强自己
5. 解锁所有成就

## 🐛 常见问题

### 游戏无法加载？
- 确保使用现代浏览器（Chrome 90+, Firefox 88+, Safari 14+）
- 使用本地服务器运行（见上方方法2）
- 检查浏览器控制台是否有错误信息

### 没有声音？
- 点击音效按钮确认没有静音
- 检查浏览器是否阻止了自动播放
- 某些浏览器需要用户交互后才能播放音频

### 游戏卡顿？
- 关闭其他占用资源的标签页
- 更新显卡驱动
- 降低浏览器缩放级别

### 成就没有解锁？
- 成就数据保存在浏览器本地存储
- 清除浏览器数据会重置成就
- 可以在成就页面查看详细进度

### 如何重置游戏？
- 访问成就页面（tank-battle-achievements.html）
- 点击"重置所有成就"按钮
- 或手动清除浏览器的localStorage

## 🎨 自定义配置

如果你想修改游戏参数，可以编辑 `src/js/tank-game/game-enhanced.js` 文件中的 `CONFIG` 对象：

```javascript
const CONFIG = {
  PLAYER: {
    SPEED: 120,        // 玩家速度
    FIRE_COOLDOWN: 0.3 // 射击冷却
  },
  ENEMY: {
    SPEED: 80,         // 敌人速度
    MAX_ACTIVE: 4      // 同时出现的敌人数
  },
  // ... 更多配置
};
```
